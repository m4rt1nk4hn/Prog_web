Bienvenue sur le meilleur site pour apprendre le français !

Ce site web est un projet pour permettre aux enfants d'acquérir les bases nécessaires à l'apprentissage de l'écriture et de la lecture
Les enfants peuvent ainsi travailler sur les syllabes et les permutations.
Il est conseillé de faire les exercices dans l'ordre, commencez donc par l'exercice 1 à gauche pour apprendre ce qu'est qu'un rébus !
